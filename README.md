# Math Challenge 🧮

Interactive console-based Math Challenge game written in C++.

---

## Description
Practice arithmetic in a fun way!  
- Choose the number of questions  
- Pick difficulty: Easy, Mid, Hard, or Mix  
- Select operation type: Add, Subtract, Multiply, Divide, or Mix  
- Program checks your answers and shows results immediately  

---

## How to Play
1. Run the program in Visual Studio or any C++ compiler.  
2. Enter how many questions you want to answer.  
3. Select the level of difficulty.  
4. Choose the type of operation or mix for variety.  
5. Answer each question and see if you’re correct.  
6. Review your final score and see if you Pass or Fail.  
7. Optionally, play again with new settings.  

---

## Features
- Randomly generated math questions  
- Difficulty levels for progressive challenge  
- Mixed operation mode for advanced practice  
- Immediate feedback with color-coded response (green = correct, red = wrong)  
- Keeps track of correct and wrong answers
